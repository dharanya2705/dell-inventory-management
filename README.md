# 🖥️ Dell Inventory Management System

A simple and interactive **Inventory Management System** built using **HTML, CSS, and JavaScript**. This project allows users to manage Dell product inventory by adding, removing, and checking product availability.

---

## 📌 Features

* ✅ Add new Dell products to inventory
* ✅ Remove products from inventory
* ✅ View current inventory list dynamically
* ✅ Check product availability (In Stock / Out of Stock)
* ✅ Clean and responsive UI

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the application
* **CSS3** – Styling and layout
* **JavaScript (Vanilla JS)** – Functionality and logic

   OR

---

## 💡 How It Works

* The inventory is stored in a **JavaScript array**.
* Users can:

  * ➕ Add products → pushes data into array
  * ❌ Remove products → removes using index
  * 🔍 Check availability → searches product in array
